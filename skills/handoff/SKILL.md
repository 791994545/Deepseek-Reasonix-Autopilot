# Handoff

Compress the current conversation into a handoff document so another agent can continue the work.

## What to include

- Current goal and status (what's done, what's next)
- Key decisions made and why
- Open questions or blockers
- Code areas touched and their current state
- Testing strategy and results so far

## Format

Keep it concise - a future agent should be able to pick up and continue within 2 minutes of reading.
