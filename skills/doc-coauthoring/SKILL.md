Trigger when user mentions writing docs, creating proposals, drafting specs, or similar documentation tasks.

<MANDATORY_EXECUTION_SCRIPT>
"=== Doc Co-Authoring Workflow ==="
""
"-- Stage 1: Context Gathering --"
"[√] Step 1: Ask meta-context (type, audience, impact, template, constraints)"
"[√] Step 2: Collect info dump from user (background, alternatives, timeline, etc.)"
"[√] Step 3: Ask 5-10 clarifying questions on gaps"
""
"-- Stage 2: Refinement & Structure --"
"[√] Step 4: Determine section structure (3-5 sections based on doc type)"
"[√] Step 5: Create initial document with placeholder text"
"[√] Step 6: For each section: clarify -> brainstorm (5-20) -> curate -> gap check -> draft -> refine iteratively"
"[√] Step 7: Near-completion review (flow, redundancy, filler, consistency)"
""
"-- Stage 3: Reader Testing --"
"[√] Step 8: Predict 5-10 realistic reader questions"
"[√] Step 9: Test with fresh sub-agent (or provide manual instructions)"
"[√] Step 10: Run additional checks (ambiguity, contradictions, assumptions)"
"[√] Step 11: Report issues, loop back to refinement if needed"
""
"=== All steps complete ==="
</MANDATORY_EXECUTION_SCRIPT>
