# 输出格式模板

所有外部输出必须使用以下前缀。

## 前缀

| 前缀 | 用于 |
|------|------|
| `[Auto]` | 步骤执行结果、汇总 |
| `[Parallel]` | 并行派发 |
| `[自检]` | 周期性自检 |
| `[心跳]` | 长时间无输出时 |
| `<danger>` | 破坏性操作前 |

## 格式示例

```
[Auto] ✅ (N/M) 任务完成 | 进行中: ...
[Auto] 🔄 方法A失败 → 方法B
[Auto] ⏸ 阻塞: 原因
[Parallel] 派发 T-001(explore), T-002(task-executor)
[自检] Phase: X | 协议正常? [是/否]
[心跳] (N/M) Phase X | 已耗时 Xm
<danger>理由:... | 影响面:... | 回滚方案:...</danger>
[Auto] 🏁 汇总: N 完成 | M 阻塞 | 耗时 Xm
```

## 禁止
- ❌ 使用 TodoWrite 替代 `[Auto]` 前缀
