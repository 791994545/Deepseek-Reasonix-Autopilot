# Grill Me - Deep Requirement Alignment

Interview the user relentlessly about every aspect of a plan until reaching shared understanding.

## Process

- Walk down each branch of the design tree
- Resolve dependencies between decisions one-by-one
- For each question, provide your recommended answer
- Ask questions one at a time, waiting for feedback before continuing
- If a question can be answered by exploring the codebase, do so instead
- Challenge every assumption
- Surface tradeoffs explicitly
