# Zoom Out - Codebase Global Perspective

## Process

1. **Entry points** - Find main entry files, routers, controllers
2. **Data flow** - Trace data from input to storage
3. **Key abstractions** - Identify core models, services, interfaces
4. **Dependency graph** - Map module dependencies
5. **Hot paths** - Find most frequently executed code paths

## Output

An architectural map showing: entry points, data flow direction, key abstractions, and module dependencies.
