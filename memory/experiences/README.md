# 经验记录 — 跨会话学习

每次 `self-improving` 复盘后的学习记录写入此目录。

## 文件格式
```
YYYY-MM-DD-<topic>.md
```

## 内容模板
```markdown
# YYYY-MM-DD: <经验主题>

## 场景
发生了什么？

## 错误的策略
尝试了什么，为什么失败？

## 正确的策略
最终怎么解决的？

## 下次遇到同类问题
直接使用什么方法/工具？
```

## 使用方式
- `self-improving` skill 复盘后自动写入
- 每次会话启动时，`self-improving` 应扫描此目录加载最近经验
